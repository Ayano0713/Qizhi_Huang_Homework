这是一个基于 **Taichi** 语言开发的 GPU 加速交互式粒子引力模拟项目，通过鼠标控制引力场，实现万级粒子的实时物理模拟与可视化。

## **一. 项目架构**

├── .venv/                             # Python 虚拟环境

├── src/Work0/                    # 核心代码包

│      ├── __init__.py                   # 包标识文件

│      ├── config.py               # 全局参数配置

│      ├── physics.py             # 物理引擎与并行计算逻辑

│      ├── main.py                 # 程序入口与渲染主循环

│      └── test.py                   # 测试脚本（可选）

├── pyproject.toml             # 项目依赖与配置（uv 管理）

├── uv.lock                          # 依赖版本锁定文件

├── .python-version          # Python 版本声明

└── README.md               # 项目说明文档



## 二. 代码逻辑

### 1. 配置层 (`config.py`)

统一管理可调参数，便于快速迭代实验：

* **物理系统**：控制粒子数量、引力强度、空气阻力与边界反弹损耗
* **渲染系统**：定义窗口分辨率、粒子绘制半径与颜色

### 2. 物理引擎层 (`physics.py`)

* **数据结构**：使用 Taichi `Vector.field` 在显存中开辟粒子位置 (`pos`) 与速度 (`vel`) 数组，支持 GPU 并行访问

* **初始化**：`init_particles()` 内核函数为每个粒子分配随机初始位置与零速度

* **物理更新**：`update_particles()` 内核函数并行计算：
  
  1. 粒子与鼠标位置的方向与距离
  2. 施加鼠标引力与空气阻力
  3. 更新粒子速度与位置
  4. 检测边界碰撞并实现能量损耗反弹

### 3. 应用层 (`main.py`)

* **Taichi 初始化**：优先调用 `ti.init(arch=ti.gpu)` 启用 GPU 加速

* **渲染循环**：
  
  1. 获取鼠标位置
  2. 调用 `update_particles()` 驱动 GPU 物理计算
  3. 将显存中的粒子位置数据拷贝至 CPU
  4. 使用 `gui.circles()` 批量绘制粒子
  5. 刷新窗口显示



三. 实现功能
----

* **GPU 并行加速**：基于 Taichi 实现万级粒子的实时物理模拟
* **交互式引力**：鼠标移动产生引力场，吸引 / 排斥粒子形成动态流体效果
* **物理特性模拟**：包含引力、空气阻力与边界碰撞反弹等真实物理效果
* **可配置性**：通过 `config.py` 快速调整粒子数量、渲染效果与物理参数
* **高效渲染**：利用 Taichi GUI 实现批量粒子绘制，保证高帧率交互体验
